In 2016 HRP (Sahel) the indicators are Gender Dis-aggregated i.e. indicator which has unit like People, Teachers etc. must provide Male/Female targets. All indicators which are not Gender Dis-Aggregated will provide one value that is Yearly Target. There are following two options which we can use to show activity targets on OPS.
