ORS - OPS Activity Targets

ORS Changes (Just for information)

For HRP 2016 (Sahel) ORS introduced few basic changes in Cluster Framework.

  1. Indicator targets at Admin level 2. In 2015 it was on Admin Level 1.
  2. Indicator must have a Unit. The Unit can be Gender-disaggregated. In case of Gender-disaggregated Cluster Coordinator has to provide Male and Female targets.
  3. Indicator must have a calculation method. This method is needed to do analysis on targeted and reported values.

OPS Changes

  1. Strategic Objective: We will only show short text it will be like 'Strategic Objective 1' etc. No change at the OPS end.
  2. Change caption 'Activity Indicator' to 'Indicator'.
  3. Add Field 'Unit'.
  4. Add Field 'Calculation Method'.
  5. Locations Changes
    1. Add double header on Locations table.
    2. Add two columns for location. 'Admin1' and 'Admin2' in place of 'Location'
    3. Add 5 more columns for target. Total columns for targets will be 6. Three for Cluster Targets and three for Project Targets.

All these chagnes/fields will be included in the XML feed. The example is given below, after table
